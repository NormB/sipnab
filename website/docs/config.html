Configuration

sipnab uses TOML configuration files. CLI flags always override config file values.

Config File Search Order

sipnab loads the first config file it finds, in this order:

  1. --config <FILE> — explicit path (must exist, error if missing)
  2. $SIPNAB_CONFIG — environment variable
  3. ~/.config/sipnab/sipnab.toml
  4. ~/.sipnabrc
  5. /etc/sipnab/sipnab.toml

Use -F / --no-config to skip loading any config file. Use -D / --dump-config to print the effective configuration.

Unknown keys produce a warning (not a hard error) to allow forward compatibility when configs are shared across versions.

Complete Example

# ~/.config/sipnab/sipnab.toml

[capture]
device = "eth0"
portrange = "5060-5080"
snaplen = 65535
buffer = 16
no_rtp = false

[display]
color = "auto"
payload_limit = 4096
delta_time = true

[filter]
from = "alice"
to = "bob"
expression = "method == 'INVITE'"

[security]
kill_scanner = true
kill_response = 403
fraud_detect = false
alert = ["syslog", "json"]
alert_exec = "/usr/local/bin/alert.sh"

[limits]
dialog_limit = 50000
max_streams = 10000
max_reassembly = 5000
hep_rate_limit = 25000

[privilege]
user = "sipnab"
no_priv_drop = false
chroot = "/var/lib/sipnab"

[theme]
background = "reset"
foreground = "white"
header = "cyan"
selected = "yellow"
accent = "magenta"
good = "green"
warning = "yellow"
bad = "red"
muted = "dark_gray"
border = "white"

[keybindings]
quit = "q"
help = "F1"
filter = "F7"
save = "F2"
search = "/"
settings = "F8"
pause = "p"
autoscroll = "A"
extended_flow = "F4"
clear_calls = "F5"
column_selector = "F10"

Section Reference

[capture]

Packet capture settings. These map to the corresponding CLI flags.

KeyTypeDefaultDescription
devicestringauto-detectDefault network interface.
portrangestring"5060-5061"SIP port range for BPF filter.
snapleninteger65535Snapshot length in bytes.
bufferintegerkernel defaultKernel buffer size in MiB.
no_rtpbooleanfalseDisable RTP capture by default.

[display]

Display and output settings.

KeyTypeDefaultDescription
colorstring"auto"Color mode: "auto", "always", or "never".
payload_limitintegerunlimitedMaximum payload bytes to display per message.
delta_timebooleanfalseShow delta time between messages by default.

[filter]

Default filter presets, applied on startup.

KeyTypeDefaultDescription
fromstring—Default From header regex filter.
tostring—Default To header regex filter.
expressionstring—Default filter DSL expression. See Filter DSL reference.

[security]

Security detection settings.

KeyTypeDefaultDescription
kill_scannerbooleanfalseEnable scanner detection on startup.
kill_responseinteger200SIP response code for scanner kill reports.
fraud_detectbooleanfalseEnable fraud detection heuristics.
alertarray of strings[]Alert channels: "syslog", "json", "exec".
alert_execstring—Command to execute on alert.

[limits]

Resource limits to prevent unbounded memory growth.

KeyTypeDefaultDescription
dialog_limitinteger100000Maximum tracked dialogs.
max_streamsinteger50000Maximum RTP streams.
max_reassemblyinteger10000Maximum TCP reassembly sessions.
hep_rate_limitinteger50000Maximum HEP packets per second.

[privilege]

Privilege dropping and sandboxing.

KeyTypeDefaultDescription
userstring—User to drop privileges to after opening capture devices.
no_priv_dropbooleanfalseDisable automatic privilege dropping.
chrootstring—Chroot directory for post-init sandboxing.

[theme]

TUI color theme. See Theme reference for details on color formats and visual previews.

KeyTypeDefaultDescription
backgroundcolor"reset"Terminal background ("reset" inherits terminal default).
foregroundcolor"white"Default text color.
highlightcolor—Legacy alias for selected.
headercolor"cyan"Status bar, column headers, endpoint labels.
selectedcolor"yellow"Selected/highlighted row, cursor, focused item.
accentcolor"magenta"Correlation info, PDD, extended flow labels.
goodcolor"green"Positive quality, success states.
warningcolor"yellow"Medium quality, caution states.
badcolor"red"Poor quality, failures, errors.
mutedcolor"dark_gray"Separators, disabled text, timestamps.
bordercolor"white"Widget borders, panel frames.

[keybindings]

TUI key binding overrides. See Keybindings reference for the full list of available keys.

KeyTypeDefaultDescription
quitkey"q"Quit the application.
helpkey"F1"Show help overlay.
filterkey"F7"Open filter dialog.
savekey"F2"Save capture.
searchkey"/"Activate search.
settingskey"F8"Open settings popup.
pausekey"p"Pause/resume capture.
autoscrollkey"A"Toggle autoscroll.
extended_flowkey"F4"Toggle extended multi-leg flow.
clear_callskey"F5"Clear all calls.
column_selectorkey"F10"Open column selector.

Key Name Syntax

Keybinding values accept these formats:

  • Single characters: "q", "/", "A"
  • Function keys: "F1" through "F12"
  • Special keys: "Esc", "Space", "Enter", "Tab", "Backspace"
