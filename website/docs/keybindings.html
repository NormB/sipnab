Keybindings

All TUI keyboard shortcuts, organized by view. Keys marked configurable can be overridden in the [keybindings] config section. Keys marked hardcoded are built into the view and cannot be remapped.

Call List

The main view. Shows all tracked SIP dialogs with sorting, filtering, and selection.

KeyAction

PgUp / PgDn Page scroll through dialog list hardcoded
Home / End Jump to first/last dialog hardcoded
Enter Open call flow for selected dialog hardcoded
Space Select/deselect dialog (for multi-select operations) hardcoded
Esc, q Quit sipnab configurable
< / > Change sort column (cycle left/right) hardcoded
Z Reverse sort direction (ascending/descending) hardcoded
A Toggle autoscroll (follow new dialogs) configurable
p Pause/resume packet capture configurable
/ Activate search (type to filter) configurable
i Clear non-matching dialogs (keep only filter matches) hardcoded
I Clear matching dialogs (invert and clear) hardcoded
t Cycle timestamp mode: absolute / delta-prev / delta-first hardcoded
F1 Show help overlay configurable
F2 Save capture (PCAP / PCAP-NG / TXT / Mermaid) configurable
F3 Search (same as /) hardcoded
F5 Clear all calls configurable
F6 Show raw SIP message hardcoded
F7 Open filter dialog configurable
F9 Clear active filter hardcoded
F10 Open column selector configurable
Tab Switch to RTP Streams view hardcoded

Call Flow

Ladder diagram view for a single dialog. Shows SIP messages as arrows between endpoints with timing, SDP detail, and a resizable detail panel.

↑ / ↓ Navigate messages (detail panel updates to show selected message) hardcoded
PgUp / PgDn Page through messages hardcoded
Home / End Jump to first/last message hardcoded
Enter Full-screen raw message view hardcoded
Space Select message for diff (press twice on different messages to compare) hardcoded
Esc Back to call list hardcoded
d Cycle SDP display: none / summary / full hardcoded
t Cycle timestamps: absolute / delta-prev / delta-first / scaled hardcoded
c Cycle arrow colors: method / call-id / cseq hardcoded
R Toggle detail panel visibility hardcoded
9 / 0, + / -, ← / → Resize ladder/detail split hardcoded
[ / ] Scroll detail panel up/down hardcoded
F2 Save capture configurable
F4, x Extended multi-leg flow (show correlated B2BUA legs) configurable
F6 Toggle RTP stream display in flow hardcoded

Raw Message

Full-screen view of a single SIP message with optional syntax highlighting and search.

KeyAction
↑ / ↓ Scroll message content hardcoded
PgUp / PgDn Page scroll hardcoded
/ Search within message hardcoded
s Toggle syntax highlighting hardcoded

Esc Back to call flow hardcoded

RTP Streams

List of all tracked RTP streams with quality metrics (jitter, loss, MOS). Accessible via Tab from the call list.

KeyAction
↑ / ↓ Navigate streams hardcoded
Tab Switch back to Call List hardcoded
F1 Show help configurable
F7 Filter streams configurable
Esc Back to Call List hardcoded

Customizing Keybindings

Override the configurable keys in your sipnab.toml:

[keybindings]
quit = "x"                # Change quit from 'q' to 'x'
help = "?"                # Change help from F1 to '?'
filter = "/"              # Change filter from F7 to '/'
save = "F2"               # Keep save at F2 (default)
search = "/"              # Keep search at '/' (default)
settings = "F8"           # Keep settings at F8 (default)
pause = "p"               # Keep pause at 'p' (default)
autoscroll = "A"          # Keep autoscroll at 'A' (default)
extended_flow = "F4"      # Keep extended flow at F4 (default)
clear_calls = "F5"        # Keep clear calls at F5 (default)
column_selector = "F10"   # Keep column selector at F10 (default)

Accepted key name formats:

  • Single characters: "q", "/", "A", "?"
  • Function keys: "F1" through "F12"
  • Special keys: "Esc", "Space", "Enter", "Tab", "Backspace"
